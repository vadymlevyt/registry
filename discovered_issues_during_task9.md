Токенів alpha немає. Залишено як inline rgba.

**Рекомендація:** в окремому TASK додати до tokens.css `color-mix()` змінні:
```css

```
Це дозволить уникнути rgba-хардкоду і зберегти зв'язок з основними кольорами при тематизації.

## Spacing — тонкі padding не округлено

Шорт-padding на кнопках/інпутах модалок:
- `'7px 10px'`, `'5px 10px'`, `'3px 9px'`, `'6px 8px'`, `'8px 10px'`, `'4px 12px'`, `'5px 12px'`, `'6px 14px'`, `'5px 14px'`

Шкала токенів:
- `--space-1: 4px`, `--space-2: 8px`, `--space-3: 12px`, `--space-4: 16px`

Округлення 5→4, 7→8, 9→8 змінить візуал на 1-2px. На око непомітно, але:
- Може порушити вертикальне вирівнювання з іконкою у кнопці
- Може зрушити загальну висоту кнопки на 2-4px (актуально якщо поряд є компонент `<Button>` з токенами — може виникнути різна висота)

**Залишено як hardcode** для сумісності з TASK 5-8 базовими `<Button size="sm">`. Окремий TASK Mobile-First CaseDossier (див. нижче) переведе на класи + токени з єдиними padding-розмірами.

## CaseDossier мобільна адаптивність — потрібен окремий TASK

**Проблема:** CaseDossier використовує **виключно inline-стилі** (без CSS класів), що унеможливлює додавання `@media` правил без рефакторингу.

Конкретні проблеми на мобільному (<768px):

1. **Шапка (header)** — рядок з кнопкою "Реєстр" + назвою справи + 5+ кнопок (Drive, Закрити, Видалити, Ідея, Агент). На <1024px не вміщається — кнопки виходять за межі екрану. Потребує:
   - Overflow меню `⋯` для другорядних кнопок
   - Або вертикальний layout

2. **Вкладки (tabs)** — 5 вкладок з іконкою + текст. На <768px тісно. Потребує:
   - Horizontal scroll (вже є в новому Tabs.css)
   - Або скорочення лейблів / прибрати текст лишити іконки

3. **Панель агента** — `width: 380px` фіксовано. На <768px:
   - Або bottom drawer
   - Або повноекранний overlay

4. **Реєстр документів двоколонковий (список + viewer)** — на портреті:
   - Список 33% + viewer 66% — тісно
   - Розв'язок: один з двох видимий за раз з перемикачем

5. **Drag-n-drop зона** — на touch не працює нативно. Потрібна:
   - Кнопка "Завантажити файли" як fallback (вже є частково)

**Estimated:** 8-12 годин окремого TASK Mobile-First CaseDossier (включає рефакторинг на CSS класи + media queries + frontload patterns).

**Передумова:** TASK 9 завершує міграцію на токени. Mobile-First TASK будує поверх токенів і додає responsive логіку.
