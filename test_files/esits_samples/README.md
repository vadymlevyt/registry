# Зразки HTML документів з ЄСІТС

Папка для зберігання реальних HTML файлів з електронного суду України (ЄСІТС) для:
- Тестування парсера pdfLibHtmlRenderer
- Snapshot тестів на регресії
- Майбутньої розробки модуля ЄСІТС
- Аналізу META даних для автоматичного імпорту справ

Адвокат буде додавати сюди різні типи документів:
- Ухвали
- Повістки
- Довідки про набрання законної сили
- Постанови
- Рішення суду
- Інші документи з електронного суду
