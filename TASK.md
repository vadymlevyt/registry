## Блок А — Навігація по тижнях

У тижневому вигляді календаря зараз немає кнопок для гортання тижнів.

**Що зробити:**
Додати кнопки ← → для переключення між тижнями прямо у тижневому вигляді.
Поведінка — така сама як кнопки місяців у місячному вигляді.
Кнопки розмістити поряд з назвою місяця ("Квітень 2026") у навігаційному рядку календаря.

При натисканні ← → у тижневому вигляді:
- переходити на попередній / наступний тиждень
- заголовок оновлювати: показувати діапазон дат тижня, наприклад "30 бер — 5 квіт"
- selectedDay оновлювати на перший день нового тижня

## Блок Б — Панель статистики під календарем

Під сіткою календаря є вільне місце. Додати туди компактну статистику.

**Що показувати:**

Рядок 1 — статуси справ:
Активних: N  ·  Призупинених: N  ·  Закритих: N

Рядок 2 — розбивка по категоріях (4 плитки в ряд):
Цивільні: N  |  Кримінальні: N  |  Військові: N  |  Адміністративні: N

Рядок 3 — заглушка білінгу:
Заголовок "💳 Білінг" і текст "Незабаром" — сірим кольором, неактивно.

**Стиль:**
- Компактно, не більше 80-90px висотою загалом
- Кольори з CSS змінних що вже є
- Цифри виділені (яскравіші), підписи приглушені
- Плитки категорій — рівномірно по ширині блоку календаря
