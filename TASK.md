## Блок А — Статистика одразу під календарем

Зараз панель статистики їде вниз за екран і не видна без прокрутки.

**Що зробити:**
Статистика має бути одразу під сіткою календаря — без прокрутки, завжди видна.
Блок календаря має займати рівно стільки висоти скільки є місця між навігацією і статистикою.
Статистика — фіксована висота внизу блоку календаря, flex-shrink: 0.

Структура блоку календаря (зверху вниз):
1. Навігація (← Квітень 2026 → | Місяць | Тиждень) — flex-shrink: 0
2. Дні тижня (Пн Вт Ср...) — flex-shrink: 0
3. Сітка днів — flex: 1, min-height: 0 (стискається якщо треба)
4. Панель статистики — flex-shrink: 0

## Блок Б — Горизонтальний бар-графік по категоріях

Замінити плитки з цифрами на горизонтальний бар-графік.

**Вигляд:**
Один рядок з кольоровими сегментами пропорційно кількості справ.
Під графіком підписи з цифрами.

Кольори сегментів:
- Цивільні: #4f7cff (синій)
- Кримінальні: #e74c3c (червоний)
- Військові: #f39c12 (помаранчевий)
- Адміністративні: #2ecc71 (зелений)

Технічно: div з display:flex, кожен сегмент flex з шириною пропорційною кількості справ.
Висота графіку: 8px, border-radius: 4px.
Під графіком рядок підписів: назва категорії кольором + цифра.

**Повна структура статистики (знизу календаря):**

Рядок 1: "Активних: 18 · Призупинених: 5 · Закритих: 0"
Рядок 2: горизонтальний бар-графік (8px висота)
Рядок 3: підписи під графіком з цифрами
Рядок 4: "💳 Білінг · Незабаром" — сірим, italic

Загальна висота статистики: не більше 80px.
