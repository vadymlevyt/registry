## 2. Поведінка ДО / ПІСЛЯ

### Searchable PDF (Рішення суду, Витяг з ЄДР, Ухвала)

| | ДО (після відкату мт5) | ПІСЛЯ (мт5.1) |
|--|------------------------|----------------|
| Перемикач Скан/Текст | прихований | прихований |
| Вміст | плашка з `<pre>{text}</pre>` (сирий extracted text) | **iframe Drive** з оригіналом |
| Виділення тексту | псевдо (selection у `<pre>`) | нативне у Drive viewer |
| Форматування | втрачене | збережене (PDF як є) |
| Текст у 02_ОБРОБЛЕНІ | створюється для агента/пошуку | створюється (без змін) |

### Scanned PDF (Адвокатський запит, Довідка садок)

| | ДО | ПІСЛЯ |
|--|-----|-------|
| Перемикач Скан/Текст | видимий | видимий |
| Скан-режим | iframe Drive | iframe Drive |
| Текст-режим | плашка з extracted text | плашка з extracted text |
| Дефолтний режим | scan (з localStorage чи 'scan') | scan (без змін) |

**Без змін.** Scanned потребує окремої Текст-плашки бо адвокат не може виділяти на самому зображенні.

## 3. Інструкція адвокату для тестування

### Сценарій А — Searchable PDF (Рішення суду, Ухвала, Витяг з ЄДР)
1. Відкрий справу з документом який вже коректно класифіковано як searchable (тобто такий що пройшов перерозпізнавання після мт3/4).
2. У досьє → вкладка Документи → відкрий документ.
3. **Очікувано:** одразу видно оригінал PDF через iframe Drive. **Без** перемикача Скан/Текст у хедері.
4. Виділи кілька рядків мишею/пальцем — Drive viewer показує контекстне меню «Копіювати», текст копіюється.
5. Прокрути сторінки скролом — багатосторінкові PDF гортаються.
6. Footer кнопки (Перерозпізнати, Обговорити з агентом) — працюють як раніше.

### Сценарій Б — Scanned PDF (Адвокатський запит з печаткою, Довідка садок)
1. Відкрий сканований документ (`documentNature='scanned'`).
2. У хедері видно перемикач Скан/Текст. Дефолт — Скан.
3. **Скан-режим:** iframe Drive — оригінал з печаткою/підписами.
4. **Текст-режим:** плашка з OCR-текстом — як раніше.
5. Перемикач зберігає вибір у localStorage (per-document).

### Сценарій В — Зображення JPG (РНОКПП, скриншот)
1. Відкрий image документ.
2. Перемикач видимий.
3. Скан-режим — `<img>` (як було).
4. Текст-режим — плашка (як було).

### Сценарій Г — DOCX (Позов, Договір)
1. Відкрий DOCX з документом.
2. Перемикача немає.
3. Видно Текст-плашку з вилученим текстом — як було.
