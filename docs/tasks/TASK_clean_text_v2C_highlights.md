## СКЛАДОВІ

### V2-C.1 — інлайн-мітки в Чистому промті (тільки `clean`)
- `buildVerbatimPrompt`: AI обгортає підозрілу/посунуту фразу маркером `==фраза==` **дослівно** (це
  позначка, не зміна слів) **І** дублює в `attentionNotes` (короткий фрагмент + причина). Делімітер `==`
  (або колізійно-безпечний — звірити, щоб не ламав GFM-таблиці/код).
- `attentionNotes`: **прибрати `page`** (ненадійний через пачкування) → форма `{ note }` у тому ж
  ПОРЯДКУ, що мітки. Оновити parse у `polishToMarkdown`, тести.
- **Тільки `clean`.** `buildDigestPrompt` (Конспект) — БЕЗ інлайн-міток (його `attentionNotes` не показуємо).

### V2-C.2 — рендер міток (MarkdownRenderer)
- `inline()`: `==x==` → `<mark class="attention" data-mark="N">x</mark>`, `N` — порядковий індекс мітки.
  На ВЖЕ екранованому тексті (XSS-safe, як `**`/`*`). Колір — **design-токен** `--attention-bg` (жовтий), НЕ hex.

### V2-C.3 — чип + панель у в'ювері (ЛИШЕ режим Чистий)
```
┌─ в'ювер, режим ЧИСТИЙ ──────────────────────────────────────┐
│  [⚠ 5 поміток]   ← ЄДИНИЙ елемент; лише коли mode==='clean'  │
└──────────────────────────────────────────────────────────────┘
        │ тап
        ▼
┌─ панель поміток ───────────────────────────────────────────┐
│  [✓] Підсвічувати в тексті        ← вкл/викл (CSS, миттєво)  │
│  1. «зникнення, знищення»  — посунув у перелік ст.170     →  │  ← клік = скрол+пульс
│  2. «…»                                                      │
│  ───────────────────────────────────────────────────────── │
│                 [ Зняти всі назавжди ]                       │
└──────────────────────────────────────────────────────────────┘
```
- Чип рендериться **лише `mode==='clean'`** (НЕ scan/exact/digest). Кількість — з `==` у `.clean.md`.
- Список: пункт ↔ `data-mark` за **порядком** (1↔1). Причини — з `extended.attentionNotes` (clean).
- Клік-навігація: `scrollIntoView({behavior:'smooth', block:'center'})` + тимчасовий клас «пульс».
- Перемикач показу: CSS-клас ховає/показує `<mark>` (миттєво, нічого не зберігаємо).
- **«Зняти всі назавжди»:** стрипнути `==` з `.clean.md` (зберегти текст) → re-save через adapter →
  очистити `attentionNotes` (clean) в extended → чип зникає.
- Спільні хелпери (parent §СПІЛЬНІСТЬ): парсинг міток, стрип `==`, scroll-to-mark — переюзні (борг #47 дорощує).

### V2-C.4 — прибрати Огляд-кнопку «Очистити тексти»
- Видалити кнопку (`CaseDossier:1866`) + `handleCleanAllTexts` (`:799`) + `services/cleanTextCycle.js`
  (+ його тести), якщо більше ніким не вживається (звірити). Масовий AI-дайджест по справі — прибрано.
- Огляд показує тексти лише як **Точний** (live, через в'ювер/наявний шлях) — нічого «запускати».

### V2-C.5 — політика місць (зафіксувати)
- В'ювер — усі режими (по одному, на вимогу). **Огляд/Реєстр — тільки Точний.** Масовий AI — гейт на сервер (parent + борг #45). Без нового коду — лише прибрати Огляд-кнопку (C.4) і не додавати реєстр-AI.
